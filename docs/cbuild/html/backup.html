备份与恢复¶

简介¶

任何人都可能碰到硬件故障和数据丢失的情况。所以定期备份长矛数据是非常关键的。因为如果面临这种情况，会导致包含非常重要且敏感数据的项目丢失数据，从而产生巨大的负面影响。 为了防止数据丢失，应定期备份长矛以便能够: * 修复长矛副本 — — 备份数据有助于修复长矛，这样就不会丢失配置设置和任何项目数据。 * 迁移不同的长矛服务器之间的数据 — — 如果遇到任何硬件更改或故障，长矛的多个实例之间的数据传输是非常有利的。
